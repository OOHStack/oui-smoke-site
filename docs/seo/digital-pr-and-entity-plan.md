# Digital PR and Entity Plan

Ethical authority building only. No PBNs, paid link schemes, fake forum posts, undisclosed advertorials, or review manipulation.

## Partnership channels

1. Wedding planners and coordinators  
2. Event agencies and corporate hospitality teams  
3. Venues (hotels, event spaces, rooftops)  
4. Photographers / videographers  
5. Caterers and F&B partners  
6. DJs and entertainment vendors  
7. Festival and brand-activation producers  

## Link targeting preference

Ask partners to link to the **most relevant service URL**, not always the homepage:

| Partner type | Preferred URL |
|--------------|---------------|
| Wedding planners | `/services/wedding-hookah-catering` |
| Corporate agencies | `/services/corporate-hookah-catering` |
| General venues | `/hookah-catering-toronto` |
| Private event hosts sharing vendors | `/services/private-event-hookah-catering` |

## Content assets that earn mentions

- Event recap galleries with accurate captions  
- Planning guides (after editorial approval)  
- Anonymized operational checklists for venues  
- Co-marketing one-pagers (`/partner` already exists)  

## Entity reinforcement

- Consistent legal name + website across citations  
- `sameAs` profiles kept accurate  
- About page + `llms.txt` + Organization schema aligned  
- Occasional expert contributions to event-planning articles (real bylines)  

## 12-month authority outline

| Quarter | Focus |
|---------|-------|
| Q1 | GBP + citations cleanup; 5 venue/planner intros; publish 2 approved guides |
| Q2 | Wedding directory profiles; 2 case studies (approved); photographer collabs |
| Q3 | Corporate agency kit; festival/hospitality partnerships |
| Q4 | Refresh guides; original seasonal planning data; review response cadence |

## Monitoring

- Brand + service SERP checks monthly  
- Referral traffic in GA4  
- Unlinked mentions → polite link requests to relevant service URLs
