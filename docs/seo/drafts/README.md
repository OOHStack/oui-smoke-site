# Guide drafts

Articles in this folder are **editorial drafts**, not published site routes.

Publishing rules:

1. Owner or editor reviews for factual accuracy  
2. No invented metrics, client names, or guarantees  
3. Answer the main question in the opening section  
4. Link to the relevant service page and `/book`  
5. Include last-reviewed date and reviewer name when published  
6. Recommend confirming venue rules and applicable laws  

See `01-how-hookah-catering-works-toronto.md` for the template style.
